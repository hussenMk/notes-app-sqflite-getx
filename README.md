Flutter Notes App & SqfLite & GetX [MVC Architecture].

## Features
- Create Note.
- Update Note.
- Delete Note.
- Delete All Notes.
- Search about notes.
- Date Time Created of Note.
- Share Note to Social Media.
- Secure data storage on SQLite.
- Languages: Arabic (ar), English (en).
- Light and Dark Modes.
- Beautiful UI.
